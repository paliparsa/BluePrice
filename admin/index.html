تنظیمات GitHub
برای انتشار خودکار روی GitHub Pages، یک Personal Access Token با دسترسی repo بسازید. این توکن فقط در مرورگر شما ذخیره می‌شود و به هیچ سرور دیگری ارسال نمی‌شود.
راهنمای راه‌اندازی اولیه
مرحله ۱: یک ریپوزیتوری جدید در GitHub بسازید (مثلاً: bluegate)
مرحله ۲: به Settings > Pages بروید و Source را روی main branch و / (root) تنظیم کنید
مرحله ۳: یک Personal Access Token بسازید: Settings > Developer settings > Personal access tokens
مرحله ۴: توکن، نام کاربری و نام ریپو را بالا وارد کنید
مرحله ۵: دکمه ذخیره و انتشار را بزنید — سایت خودکار آپلود می‌شود!
